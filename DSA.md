# DSA — Start Here

The full study guide lives in the [`dsa-mastery/`](./dsa-mastery/) folder.

### Prefer a nicer read? Open the HTML site

Double-click [`dsa-mastery/site/index.html`](./dsa-mastery/site/index.html) to open the
styled, searchable site in your browser — syntax highlighting, sidebar navigation,
on-page table of contents, copy-code buttons, and a light/dark theme toggle. It works
fully offline (libraries are vendored locally). Re-generate it any time you edit the
Markdown with `python3 dsa-mastery/build_site.py`.

### Or read the Markdown directly

Open [`dsa-mastery/README.md`](./dsa-mastery/README.md) for the roadmap, the master
mnemonic, and the complexity index, then work through the files in order:

1. [`00-complexity-foundations.md`](./dsa-mastery/00-complexity-foundations.md) — Big-O ladder + data-structure cheat sheet.
2. [`01-core-patterns.md`](./dsa-mastery/01-core-patterns.md) — the 15 core LeetCode patterns (Python).
3. [`02-dynamic-programming.md`](./dsa-mastery/02-dynamic-programming.md) — the 20 dynamic programming patterns (Python).
4. [`03-flashcards.md`](./dsa-mastery/03-flashcards.md) — rapid recall + spaced-repetition schedule.

Based on AlgoMaster's
[15 LeetCode Patterns](https://blog.algomaster.io/p/15-leetcode-patterns) and
[20 Patterns to Master Dynamic Programming](https://blog.algomaster.io/p/20-patterns-to-master-dynamic-programming),
expanded with diagrams, mnemonics, complexity tables, and worked Python examples.
