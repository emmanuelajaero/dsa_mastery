# 05 — Progress Tracker

> **How to use:** Copy this file or edit it directly. Change `☐` to `☑` as you
> complete each step. The columns follow the recommended study loop:
>
> 1. **Read** the pattern section and understand the template.
> 2. **Re-derive** — cover the code and write the template from memory.
> 3. **LeetCode 1** — solve the first practice problem.
> 4. **LeetCode 2** — solve a second practice problem.
> 5. **Day 1** — flashcard recall (Deck A + B) from memory.
> 6. **Day 3** — flashcard recall (Deck C recurrences) + solve another problem.
> 7. **Day 7** — concept checks (Deck D) + mixed random review.
>
> **Golden rule:** If you can write the template from a blank editor and state its
> complexity without looking, mark it done.
